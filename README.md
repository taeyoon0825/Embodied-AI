# Embodied AI 소개 페이지

Embodied AI를 처음 접하는 사람도 쉽게 이해할 수 있도록 만든 단일 스크롤 랜딩 페이지입니다.  
개념 설명부터 작동 방식, 실제 사례, 핵심 기술, 중요성, 도전 과제까지 한 페이지에서 자연스럽게 이해할 수 있도록 구성했습니다.

## 프로젝트 목적

- AI 비전공자/초보자/일반인 대상의 쉬운 설명 제공
- 논문식 문체보다 읽기 쉬운 소개형 콘텐츠 중심 구성
- `Embodied AI가 무엇인지`를 남에게 설명할 수 있는 수준의 이해 지원

## 파일 구조

```text
Embodied AI/
├─ index.html   # 페이지 구조 및 콘텐츠
├─ style.css    # 레이아웃, 색상, 반응형, 카드/호버 스타일
├─ script.js    # 스크롤 기반 fade-in 애니메이션
└─ README.md
```

## 포함된 섹션

1. 히어로 섹션  
2. Embodied AI 개념 설명  
3. 작동 과정(Perception → Understanding → Planning → Action)  
4. 실제 예시(자율주행차, 물류 로봇, 가정용 서비스 로봇, 시각장애인 보조 AI)  
5. 활용 기술(컴퓨터 비전, NLP/LLM, 강화학습, 로봇 제어, 경로 계획, 센서 융합)  
6. 왜 중요한가  
7. 한계와 어려움(도전 과제 관점)  
8. 마무리 메시지  

## 실행 방법

별도 빌드 도구나 라이브러리 없이 실행 가능합니다.

1. 프로젝트 폴더를 엽니다.
2. `index.html` 파일을 브라우저에서 실행합니다.

또는 VS Code/Cursor의 Live Server 확장을 사용해 실행할 수 있습니다.

## 디자인/구현 특징

- 반응형 레이아웃(모바일/태블릿/데스크톱 대응)
- 카드형 UI + 아이콘 느낌 요소 + hover 효과
- 미래지향적이지만 과하지 않은 다크 톤 그라데이션
- 섹션 간 충분한 여백으로 가독성 확보
- `IntersectionObserver` 기반 fade-in 스크롤 애니메이션
- `prefers-reduced-motion` 환경 고려(모션 축소 시 애니메이션 최소화)

## 커스터마이징 가이드

### 1) 색상 테마 변경

`style.css`의 `:root` 변수 값을 수정하면 전체 톤을 빠르게 변경할 수 있습니다.

- `--bg`, `--surface`: 배경/카드 배경
- `--primary`, `--accent`: 포인트 컬러
- `--text-main`, `--text-muted`: 본문/보조 텍스트

### 2) 콘텐츠 문구 수정

- 주요 텍스트는 `index.html` 각 섹션에 직접 작성되어 있습니다.
- 발표/소개 대상에 맞춰 문장 길이와 사례를 조정하면 좋습니다.

### 3) 애니메이션 강도 조절

- `script.js`의 `threshold`, `rootMargin` 값을 변경해 표시 타이밍을 조절할 수 있습니다.
- `style.css`의 `.reveal` transition 시간을 변경해 속도를 조절할 수 있습니다.
